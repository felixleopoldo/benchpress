.. rubric:: Description

Independently identically distributed (IID) samples of a given size ``n``.
The data may be standardized by setting ``standardized`` to true. 




.. rubric:: Example


.. code-block:: json


    [
      {
        "id": "standardized",
        "standardized": true,
        "n": [
          100
        ]
      }
    ]
