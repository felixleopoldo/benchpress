.. rubric:: Description

An object of the max bandwith module specifies a decomposable graph with band structured adjacency matrix of a given maximum width (``max_bandwith``).


.. rubric:: Example


.. code-block:: json


    [
      {
        "id": "AR1-5",
        "max_bandwidth": 5,
        "dim": 50
      }
    ]
